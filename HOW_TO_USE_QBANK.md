## üîÑ **CLONE vs MOVE:**

### **CLONE (Drag & Drop):**
- ‚úÖ Creates a **copy** of the question
- ‚úÖ Original stays in place
- ‚úÖ Question exists in **multiple folders**
- ‚úÖ Good for: Reusing questions across topics

**Example:**
```
Drag "Cardiac Assessment" from All Questions
Drop on "Adult Health"
Drop on "Critical Care"
Result: Question in 3 places (All, Adult, Critical)
```

### **MOVE (Dropdown):**
- ‚úÖ Changes **primary folder**
- ‚úÖ Question belongs to **one folder**
- ‚úÖ Updates the "FOLDER" column
- ‚úÖ Good for: Organizing and categorizing

**Example:**
```
Question shows "None" in dropdown
Change to "Adult Health"
Result: Question's primary folder is now Adult Health
```
